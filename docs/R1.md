# Sentence-Pruning Dataset Objective

This repo creates local transition sentence-pruning data for reasoning tasks. The runner generates model reasoning, asks a live LiteLLM decision model for a safely removable contiguous sentence span, verifies the shortened context, and saves accepted training records.

Run the active workflow with:

```bash
uv run python scripts/create_pruning_dataset.py --config config/default.toml
```

Dataset-specific runs use the same command with another config under `config/*.toml`. Prompts, source settings, model settings, output paths, quality gates, and upload settings live in config.

Accepted records use the local transition shape: `input_x` is the question/current context plus the useful prefix, `target_y` is the next useful sentence after the skipped span, and metadata records the removed span, quality checks, source task, model settings, and `format_version`.
